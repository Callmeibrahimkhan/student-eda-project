# Student Dataset - Exploratory Data Analysis (EDA)

## 📖 Project Overview
This project demonstrates the fundamentals of Exploratory Data Analysis (EDA) using Python, Pandas, and NumPy on a student dataset.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## 📊 EDA Functions Used
- head()
- tail()
- shape
- columns
- info()
- describe()
- isnull().sum()
- value_counts()
- mean()
- max()
- min()

## 📁 Project Files
- Student_EDA.ipynb
- students.csv

## 🎯 Learning Outcome
This project helped me understand:
- Reading CSV files
- Exploring datasets
- Statistical summary
- Missing value detection
- Basic data analysis with Pandas
